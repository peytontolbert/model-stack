## metrics

Mask-aware metrics and calibration utilities.

### Classification metrics
- `masked_accuracy`, `masked_topk_accuracy`, `masked_token_f1`, `masked_span_f1`

### Calibration/uncertainty
- `ece_binning`, `ece_temperature_sweep`, `plascale_logits`, `vector_scale_logits`, `brier_score_masked`

### Sequence metrics
- `sequence_logprob`, `sequence_entropy`, `distinct_n`, `self_bleu`, `uniq_ngrams`
