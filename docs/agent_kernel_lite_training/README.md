# Agent Kernel Lite Training Docs

This directory contains the source planning docs imported from
`/data/transformer_10` for model-stack migration.

Included topics:

- F5TTS WASM/iPhone feasibility and Q4 deployment planning
- FLUX tiny student distillation findings
- PocketPal seq2seq supervision and training plans
- agentic dataset formats and AK token protocol notes
- training refinement notes for controller/assistant behavior

These documents are historical source material plus migration guidance. They are
not all implemented model-stack contracts yet. Promote a document into a
package-level README or a top-level `docs/` contract only after its scripts and
runtime paths have been adapted to model-stack naming and tests.
